# 长夜书藏 / EnLib

A card-based librarian game set in the Eternal Night world.
Built with Godot 4.6 + C# (.NET 8).

See [DESIGN.md](DESIGN.md) for the design doc.

## Setup

1. Install [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
2. Download [Godot 4.6 .NET](https://godotengine.org/download)
3. Open `project.godot` in Godot.
4. On first open, Godot will create `.godot/` and build the C# project.

## Run

- Press F5 in the Godot editor, or
- `Godot_v4.6.x-stable_mono_win64.exe --path . scenes/Main.tscn`
